# 學習 SQL 系列 - 什麼是 SQL?

- SQL 全名為 Structured Query Language，結構化==查詢==語言。
- 透過 SQL 語法可以存取(access)與操作(manipulate)資料庫(database)。
- SQL 是一種標準(standard)，而各家服務(資料庫程式，如 MySql)基於這個標準皆有自己額外的擴充。

## SQL 可以做到什麼事情？

- SQL 可以對資料庫執行==查詢(query)==
- SQL 可以在資料庫中 ==插入(insert)== 記錄(record)
- SQL 可以 ==更新(update)== 資料庫中的記錄
- SQL 可以從資料庫中 ==刪除(delete)== 記錄
- SQL 可以==建立(create)新的資料庫==
- SQL 可以在資料庫中==建立新表==
- SQL 可以在資料庫中==建立預存程序==
- SQL 可以設定==表格、流程和檢視的權限(permission)==

## RDBMS (Relational Database Management System)

- 關連式資料庫，由直欄(column)、橫列(row)組成一個表格(table)，可以預先定義資料所需要的資訊，並用來儲存資料於此。
- 每一橫列可以透過一個主索引鍵(Primary key)作為唯一識別符，表格之間可以透過外部索引鍵(Foreign key)建立關聯。
