## 活用：

示意圖：![](https://i.imgur.com/0G74DVN.png)

1. 狀況ㄧ：只顯示d區塊
    ```html
    <div class="box"></div>
    ```
    ```html
    .box{
        width:0px;
        height:0px;
        border-style:solid;
        border-color: transparent transparent transparent #dad;
        border-width:80px;
    }
    ```

---

2. 狀況二：顯示b+c區塊
    ```html
    <div class="box"></div>
    ```
    ```html
    .box{
            width:0px;
            height:0px;
            border-style:solid;
            border-color: transparent #ada #dda transparent;
            border-width:80px;
    }
    ```
